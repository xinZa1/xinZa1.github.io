PHP 5 和 PHP 7 的区别

在 PHP 5 中，assert() 是一个函数，我们可以用 $=assert;$() 这样的形式来实现代码的动态执行。但是在 PHP 7 中，assert() 变成了一个和 eval() 一样的语言结构，不再支持上面那种调用方法。（但是好像在 PHP 7.0.12 下还能这样调用）
PHP5中，是不支持 ($a)() 这种调用方法的，但在 PHP 7 中支持这种调用方法，因此支持这么写 (‘phpinfo’)();
